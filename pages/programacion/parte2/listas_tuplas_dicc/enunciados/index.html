Ejercicios: Listas, Tuplas y Diccionarios en Python 📖💻

1️⃣ Ordenar una lista

1. Escribe una función llamada ordenar_lista que reciba una lista de números que devuelva la lista ordenada en orden ascendente.

2. Modulariza el programa importando la función desde un archivo llamado utilidades.py.

Ejemplo: lista = [5, 3, 8, 1, 2]

lista_ordenada = ordenar_lista(lista)

print(lista_ordenada) # Salida: [1, 2, 3, 5, 8]

2️⃣ Filtrar números pares

1. Escribe una función que reciba una lista de números y retorne una nueva lista con solo los números pares.

2. Utiliza la función dentro del archivo principal, pero crea el módulo filtrado.py para la definición de la función.

Ejemplo: lista = [1, 2, 3, 4, 5, 6, 7, 8, 9]

pares = filtrar_pares(lista)

print(pares) # Salida: [2, 4, 6, 8]

3️⃣ Eliminar duplicados de una lista

1. Crea una función llamada eliminar_duplicados que reciba una lista de números con elementos duplicados y retorne una nueva lista sin duplicados.

2. Obliga a que la función esté definida en un módulo externo llamado listas_util.py.

Ejemplo: lista = [1, 2, 2, 3, 4, 4, 5]

nueva_lista = eliminar_duplicados(lista)

print(nueva_lista) # Salida: [1, 2, 3, 4, 5]

4️⃣ Desempaquetar tuplas

1. Crea una función llamada desempaquetar_tupla que reciba una tupla con tres elementos y devuelva cada uno de ellos por separado.

2. La función debe estar en el archivo principal.

Ejemplo: tupla = (10, 20, 30)

a, b, c = desempaquetar_tupla(tupla)

print(a, b, c) # Salida: 10 20 30

5️⃣ Crear una lista de tuplas

1. Escribe un programa que cree una lista de tuplas, donde cada tupla contenga el índice y el valor de cada elemento de una lista original.

2. Define una función llamada crear_lista_tuplas y colócala en un módulo externo llamado tuplas_util.py.

Ejemplo: lista = ['a', 'b', 'c', 'd']

lista_tuplas = crear_lista_tuplas(lista)

print(lista_tuplas) # Salida: [(0, 'a'), (1, 'b'), (2, 'c'), (3, 'd')]

6️⃣ Contar elementos en una tupla

1. Crea una función llamada contar_elementos_tupla que reciba una tupla y devuelva la cantidad de veces que aparece un elemento específico.

2. Obliga a que la función esté en el archivo tuplas_util.py.

Ejemplo: tupla = (1, 2, 3, 1, 2, 1)

cuenta = contar_elementos_tupla(tupla, 1)

print(cuenta) # Salida: 3

7️⃣ Crear diccionario a partir de dos listas

1. Crea una función llamada crear_diccionario que reciba dos listas: una de claves y otra de valores, y las combine en un diccionario.

2. Importa esta función desde el archivo diccionarios_util.py.

Ejemplo: claves = ['nombre', 'edad', 'ciudad']

valores = ['Carlos', 25, 'Madrid']

diccionario = crear_diccionario(claves, valores)

print(diccionario) # Salida: {'nombre': 'Carlos', 'edad': 25, 'ciudad': 'Madrid'}

8️⃣ Contar ocurrencias de palabras

1. Escribe un programa que cuente cuántas veces aparece cada palabra en una cadena de texto y almacene los resultados en un diccionario.

2. La lógica para contar las palabras debe estar dentro de una función llamada contar_palabras, ubicada en un módulo texto_util.py.

Ejemplo: texto = "hola hola adios hola"

conteo = contar_palabras(texto)

print(conteo) # Salida: {'hola': 3, 'adios': 1}

9️⃣ Revertir un diccionario

1. Escribe una función llamada revertir_diccionario que invierta un diccionario, es decir, que los valores se conviertan en las claves y las claves en los valores.

2. Coloca la función en el módulo diccionarios_util.py.

Ejemplo: diccionario = {'a': 1, 'b': 2, 'c': 3}

diccionario_invertido = revertir_diccionario(diccionario)

print(diccionario_invertido) # Salida: {1: 'a', 2: 'b', 3: 'c'}

9️⃣9️⃣ Proyecto Modularizado

1. Crea un pequeño proyecto donde deberás implementar las siguientes funcionalidades:

- Crear una lista de números aleatorios entre 1 y 100.

- Encontrar el número más grande y el más pequeño de la lista.

- Contar cuántas veces aparece cada número en la lista y almacenar los resultados en un diccionario.

- Ordenar la lista de manera ascendente.

2. Estructura del proyecto:

- Archivo main.py: El archivo principal que importará las funciones necesarias.

- Módulo listas_util.py: Contendrá las funciones para generar la lista, encontrar el máximo y mínimo.

- Módulo diccionarios_util.py: Contendrá las funciones para contar las ocurrencias de los números.

- Módulo ordenar.py: Contendrá la función para ordenar la lista.
